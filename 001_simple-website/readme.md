# Simple website template

HTMLとCSSによるウェブサイト テンプレートを0から作成する動画で使用したサンプルです. 

動画: https://youtu.be/KCIxflGXgaU



## 実装内容

- 全画面画像
- 全画面動画
- タイル表示
- 2カラム, 3カラム構成 (聖杯レイアウト) の FlexBox or Gridでの実装



## 作成環境

websiteなので基本的には環境に依存しないはずです. 

下記は, 作成時の私の環境です. 参考まで. 

- Macbook Air M1 2020
- Editor: VS Code
- Browser: Chrome (, Safari)
